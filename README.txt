Para ejecutar el proyecto es necesario seguir estos pasos:

	1. Descargar node.js
	2. Agregar node.js a las variables de entorno 
	3. Abrir la consola de node.js e ir al directorio donde se encuentra el proyecto
	4. a través del comando npm install, descargar las siguientes librerias:	
		body-parser	
		cookie-parser
		debug
		express	
		jade
		mongoose
		morgan
		multer	
		serve-favicon
		xls-to-json-lc
		xlsx-to-json-lc
	5.Verificar que las librerias se hayan instalado en la carpeta node modules
	6. Ejecutar el proyecto con el comando
		node bin/www
